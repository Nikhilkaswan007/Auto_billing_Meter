Energy Meter API Documentation

Add Reading Endpoint

Endpoint:

POST /api/reading/add/

Request Body (JSON):

{
    "device_id": "your_device_id",
    "kilowatt_hours": 15.5,
    "voltage": 240.0,
    "ampere": 10.5
}

Success Response (200):

{
    "status": "success",
    "message": "Reading saved successfully",
    "reading_id": 1
}

Error Response (404 - Meter not found):

{
    "status": "error",
    "message": "Meter not found"
}

Python Example (Raspberry Pi)

import requests
import json

url = 'http://your-server-ip:8000/api/reading/add/'

data = {
    'device_id': 'METER_001',
    'kilowatt_hours': 15.5,
    'voltage': 240.0,
    'ampere': 10.5
}

response = requests.post(url, json=data)
print(response.json())
Make sure the device_id exists in the database before sending readings!
